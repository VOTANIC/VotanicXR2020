VotanicXR 2020 API Reference

A toolkit for creating VR / XR application in CAVE System

VotanicXR software requirement

The VotanicXR Software Development Kit (SDK) can be used for free for an unlimited time upon acquiring and activating the evaluation (trial) license. A professional license is required for building a commercial project with the VotanicXR SDK. Academic License is available for education and non-profit projects. Licenses will default to a trial license if this license is absent. Projects built under a trial license will have a watermark and limited to 1 minute of running time.

The following software are required for using VotanicXR:

  • Unity 2018.2 or above
    • Unity 2018 is recommended for best compatibility
  • Visual C++ Redistributable 2012
  • Windows 7 SP3 or above (Windows 10 build 1909 or above Preferred)
  • Visual Studio 2017 or above
  • End User Hardware License
    • The End User Hardware License is required for running application on Cave Automatic Virtual Environment (CAVE) or DOME system. The default single camera system configuration for desktop system will be used if this hardware license is absent.
  • SteamVR
    • Steam VR software from Steam is needed if you wish to use the OpenVR Plugin or build applications for the Head-mounted Display (HMD) system. Download and install the Steam VR software at https://store.steampowered.com/app/250820/SteamVR/
  • Oculus Software
    • The Oculus software for any Oculus HMD is needed if you wish to use an Oculus HMD device. Download and install the software at https://www.oculus.com/setup/
  • Windows Mixed Reality for SteamVR
    • SteamVR and Windows Mixed Reality plugin on SteamVR is needed if you wish to use Windows Mixed Reality device. Download andinstall the software at https://store.steampowered.com/app/719950/Windows_Mixed_Reality_for_SteamVR/
